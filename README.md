# Keyflow Typing Practice

A static, client-only typing practice website built with Vite + React + TypeScript.

## What it includes

- Local-only typing rounds with no server or backend cost.
- Multiple practice modes and drill types.
- Saved session history in browser localStorage.
- Ad-ready layout with placeholder slots for future monetization.
- Responsive UI designed to work on free static hosting.

## Run locally

```bash
npm install
npm run dev
```

## Build for deployment

```bash
npm run build
```

The output in `dist/` can be deployed to GitHub Pages, Cloudflare Pages, Netlify, or any other static host.
